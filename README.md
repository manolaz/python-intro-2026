## üöÄ Getting Started

### Prerequisites

- Python 3.x installed on your computer
- A text editor or IDE (VS Code, PyCharm, or IDLE)
- Basic computer literacy

### Installation

1. **Install Python:**
   - Download from [python.org](https://python.org)
   - Follow the installation instructions for your operating system
   - Verify installation: `python --version`

2. **Clone this repository:**
   ```bash
   git clone https://github.com/manolaz/python-intro-2026.git
   cd python-intro-2026
   ```

3. **Start learning:**
   - Read the lessons in order starting with Lesson 1
   - Complete the exercises after each lesson
   - Check your solutions against the provided answers

## üìñ How to Use This Course
