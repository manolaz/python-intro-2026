# Python Introduction Course - Syllabus

## Course Information

**Course Name:** Introduction to Python Programming 2026  
**Level:** Beginner  
**Duration:** 6 weeks (self-paced)  
**Prerequisites:** None  

## Course Description

This comprehensive introductory course is designed to teach the fundamental concepts of programming using Python. Students will learn Python syntax, basic programming constructs, data structures, and file operations through a combination of theoretical lessons, practical exercises, and hands-on projects.

By the end of this course, students will have a solid foundation in Python programming and will be able to write basic programs to solve real-world problems.

## Learning Objectives

Upon successful completion of this course, students will be able to:

1. Understand and apply Python syntax and programming fundamentals
2. Use variables, data types, and operators effectively
3. Implement control flow using conditional statements and loops
4. Design and use functions to organize code
5. Work with various data structures (lists, tuples, dictionaries, sets)
6. Perform file input/output operations
7. Debug basic Python programs
8. Write clean, readable, and well-documented code

## Course Outline

### Week 1: Introduction and Basics

**Lesson 1: Introduction to Python**
- What is Python and why learn it?
- Installing Python and setting up the environment
- Writing your first Python program
- Understanding Python syntax
- Comments and code documentation
- Basic input/output operations

**Topics Covered:**
- Print statements
- Input function
- Comments (single and multi-line)
- Python indentation rules

**Practice:** 5 exercises focusing on basic syntax and I/O

### Week 2: Variables and Data Types

**Lesson 2: Variables and Data Types**
- Variable declaration and naming conventions
- Numeric types (integers, floats)
- String manipulation and methods
- Boolean values and logic
- Type conversion and type checking
- Basic operators (arithmetic, comparison)

**Topics Covered:**
- Variable assignment
- String operations and formatting
- Arithmetic operations
- Type casting
- Boolean operations

**Practice:** 10 exercises on variables, operators, and data types

---

### Week 3: Control Flow

**Lesson 3: Control Flow**
- Conditional statements (if, elif, else)
- Logical operators (and, or, not)
- For loops and iteration
- While loops
- Loop control (break, continue)
- Nested loops and conditions

**Topics Covered:**
- Decision making with if statements
- Iterating with for loops
- Conditional loops with while
- Pattern creation with nested loops
- FizzBuzz and other classic problems

**Practice:** 13 exercises including pattern printing and games

---

### Week 4: Functions

**Lesson 4: Functions**
- Defining and calling functions
- Parameters and arguments
- Return values
- Default parameters
- Multiple return values

- Lambda functions
- Built-in functions

**Topics Covered:**
- Function definition and syntax
- Passing arguments
- Returning values
- Function documentation
- Anonymous functions (lambda)
- Practical function applications

**Practice:** 15 exercises on function creation and usage

---

### Week 5: Data Structures

**Lesson 5: Data Structures**
- Lists: creation, indexing, slicing
- List methods and operations
- List comprehensions
- Tuples and immutability
- Dictionaries: key-value pairs
- Dictionary operations
- Sets and set operations
- Nested data structures

**Topics Covered:**
- Working with collections
- Choosing the right data structure
- Data manipulation techniques
- Comprehensions for concise code
- Common data structure patterns

**Practice:** 18 exercises on lists, dictionaries, tuples, and sets

---

### Week 6: File Operations

**Lesson 6: File Operations**
- Reading from files
